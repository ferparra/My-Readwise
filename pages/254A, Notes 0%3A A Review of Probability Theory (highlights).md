	- We list some important examples of discrete distributions:
	  
	  •   [Dirac distributions](http://en.wikipedia.org/wiki/Dirac_delta_function) ![{\delta_{x_0}}](https://s0.wp.com/latex.php?latex=%7B%5Cdelta_%7Bx_0%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), in which ![{p_x = 1}](https://s0.wp.com/latex.php?latex=%7Bp_x+%3D+1%7D&bg=ffffff&fg=000000&s=0&c=20201002) for ![{x=x_0}](https://s0.wp.com/latex.php?latex=%7Bx%3Dx_0%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{p_x=0}](https://s0.wp.com/latex.php?latex=%7Bp_x%3D0%7D&bg=ffffff&fg=000000&s=0&c=20201002) otherwise;
	  •   [discrete uniform distributions](http://en.wikipedia.org/wiki/Uniform_distribution_(discrete)), in which ![{R}](https://s0.wp.com/latex.php?latex=%7BR%7D&bg=ffffff&fg=000000&s=0&c=20201002) is finite and ![{p_x = 1/|R|}](https://s0.wp.com/latex.php?latex=%7Bp_x+%3D+1%2F%7CR%7C%7D&bg=ffffff&fg=000000&s=0&c=20201002) for all ![{x \in R}](https://s0.wp.com/latex.php?latex=%7Bx+%5Cin+R%7D&bg=ffffff&fg=000000&s=0&c=20201002);
	  •   (Unsigned) [Bernoulli distributions](http://en.wikipedia.org/wiki/Bernoulli_distribution), in which ![{R=\{0,1\}}](https://s0.wp.com/latex.php?latex=%7BR%3D%5C%7B0%2C1%5C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), ![{p_1=p}](https://s0.wp.com/latex.php?latex=%7Bp_1%3Dp%7D&bg=ffffff&fg=000000&s=0&c=20201002), and ![{p_0=1-p}](https://s0.wp.com/latex.php?latex=%7Bp_0%3D1-p%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some parameter ![{0 \leq p \leq 1}](https://s0.wp.com/latex.php?latex=%7B0+%5Cleq+p+%5Cleq+1%7D&bg=ffffff&fg=000000&s=0&c=20201002);
	  •   The *signed Bernoulli distribution*, in which ![{R = \{-1,+1\}}](https://s0.wp.com/latex.php?latex=%7BR+%3D+%5C%7B-1%2C%2B1%5C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{p_{+1}=p_{-1} = 1/2}](https://s0.wp.com/latex.php?latex=%7Bp_%7B%2B1%7D%3Dp_%7B-1%7D+%3D+1%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002);
	  •   *Lazy signed Bernoulli distributions*, in which ![{R = \{-1,0,+1\}}](https://s0.wp.com/latex.php?latex=%7BR+%3D+%5C%7B-1%2C0%2C%2B1%5C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002), ![{p_{+1}=p_{-1}=\mu/2}](https://s0.wp.com/latex.php?latex=%7Bp_%7B%2B1%7D%3Dp_%7B-1%7D%3D%5Cmu%2F2%7D&bg=ffffff&fg=000000&s=0&c=20201002), and ![{p_0 = 1-\mu}](https://s0.wp.com/latex.php?latex=%7Bp_0+%3D+1-%5Cmu%7D&bg=ffffff&fg=000000&s=0&c=20201002) for some parameter ![{0 \leq \mu \leq 1}](https://s0.wp.com/latex.php?latex=%7B0+%5Cleq+%5Cmu+%5Cleq+1%7D&bg=ffffff&fg=000000&s=0&c=20201002);
	  •   [Geometric distributions](http://en.wikipedia.org/wiki/Geometric_distribution), in which ![{R = \{0,1,2,\ldots\}}](https://s0.wp.com/latex.php?latex=%7BR+%3D+%5C%7B0%2C1%2C2%2C%5Cldots%5C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{p_k = (1-p)^k p}](https://s0.wp.com/latex.php?latex=%7Bp_k+%3D+%281-p%29%5Ek+p%7D&bg=ffffff&fg=000000&s=0&c=20201002) for all natural numbers ![{k}](https://s0.wp.com/latex.php?latex=%7Bk%7D&bg=ffffff&fg=000000&s=0&c=20201002) and some parameter ![{0 \leq p \leq 1}](https://s0.wp.com/latex.php?latex=%7B0+%5Cleq+p+%5Cleq+1%7D&bg=ffffff&fg=000000&s=0&c=20201002); and
	  •   [Poisson distributions](http://en.wikipedia.org/wiki/Poisson_distribution), in which ![{R = \{0,1,2,\ldots\}}](https://s0.wp.com/latex.php?latex=%7BR+%3D+%5C%7B0%2C1%2C2%2C%5Cldots%5C%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) and ![{p_k = \frac{\lambda^k e^{-\lambda}}{k!}}](https://s0.wp.com/latex.php?latex=%7Bp_k+%3D+%5Cfrac%7B%5Clambda%5Ek+e%5E%7B-%5Clambda%7D%7D%7Bk%21%7D%7D&bg=ffffff&fg=000000&s=0&c=20201002) for all natural numbers ![{k}](https://s0.wp.com/latex.php?latex=%7Bk%7D&bg=ffffff&fg=000000&s=0&c=20201002) and some parameter ![{\lambda}](https://s0.wp.com/latex.php?latex=%7B%5Clambda%7D&bg=ffffff&fg=000000&s=0&c=20201002).
